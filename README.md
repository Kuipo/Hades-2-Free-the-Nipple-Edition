# NSFW Resprite

This mod replaces most of the 2D artwork in the game with NSFW images.

This is a NSFW mod for Hades 2. This is currently for The 1.1 release. This replaces some of the in-game 2D artwork with NSFW images. NOTE: You must be using high resolution textures to see the replacement images.

### Disclaimer:

This mod is made using hand edited images by eddyboy1337 and coded by Kuipo.

## Features

- Replaces 14 character portraits for the female characters in the game
- Modifies some of the effects such as glint, glow, and wiggle where appropriate to adjust for the changes in the armor

## Install

- Install r2modman
- Enable the Show NSFW content in r2modman filter
- Install this mod and it's dependancies
